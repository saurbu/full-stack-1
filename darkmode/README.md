# 🌗 Dark Mode Toggle (HTML, CSS, JavaScript)

A simple web project that demonstrates how to implement a **Light Mode / Dark Mode toggle** using **JavaScript, CSS, and localStorage**.  
The selected theme is saved in the browser so that it remains the same even after refreshing the page.

---

## 🚀 Features

- 🌞 Light Mode and 🌙 Dark Mode toggle
- 💾 Theme preference saved using **localStorage**
- ⚡ Instant UI updates without reloading the page
- 🎨 Smooth color transition between themes
- 🧠 Displays the current `localStorage` value on the page

---

## 🛠 Technologies Used

- **HTML5** – Structure of the webpage  
- **CSS3** – Styling and dark mode theme  
- **JavaScript (Vanilla JS)** – Toggle functionality and localStorage handling
